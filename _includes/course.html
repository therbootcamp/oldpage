R Course

An in-depth course for beginners, academics, and professionals. We cover classic frequentist and modern Bayesian statistics. We show you how to make elegant, informative data visualizations. We introduce you to some of R's most efficient libraries for data analysis and tools to write transparent, reproducible reports. We demonstrate big data applications and cutting-edge machine-learning techniques.

Essentials


Get started with R and RStudio. Install software and packages. Get your data into and out of R and doing basic manipulations.

Statistics


Descriptive and inferential statistics. T-tests, ANOVA, regression. Bayesian statistics. Machine learning and statistical prediction.

Visualisation


Histograms, barplots, scatterplots -- even our own pirateplots. Networks and maps. Interactive graphics with Shiny.

Productivity


Write, organize, and share reproducible code using R Markdown and GitHub. Speed up your code with C++ and parallel programming.
